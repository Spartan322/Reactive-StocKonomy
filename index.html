    Welcome to Reactive StocKonomy's Official Site.

    A Minecraft Bukkit Plugin

    This Minecraft Bukkit Plugin is meant to improve a server's feel by adding an Economy and a Stock Market Plugin that isn't going to be abandoned. The best part is it can be independent.

    Contributing

    To best help our project, contribute To contribute just...

    $ cd (the folder to hold your code)
$ git clone https://github.com/Spartan322/Reactive-StocKonomy.git
$ git add --all (when you are done messing with the code)
$ git commit -m "[name]" (the name is what you should call the commit)
$ git push (to push back to the Github)

    Use your Username and password to put it in a fork then when your are done just pull to the Main one from Github itself

    Rules to Contributing

    • Please use organized code.
    • Use only Eclipse's Wrap. Github is not as great
    • Eclipse would be the best idea
    • Please keep the functions, variables, and imports organized
    • Keep Files short for easier read (150 lines is pretty long if you can shorten it it would be great)
